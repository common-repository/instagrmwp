== Description ==

InstagrmWP is a plugin for everybody like photo from Instagram and want buid-in wordpress. this plugin display Tab, Information, Feed photo from admin of InstagrmWP and setting on admin page. This plugin requires PHP curl extension. 


Demo InstagrmWP plugin by wordpress default ( "WP_DEBUG", true ) <a href="http://www.wpsocialmediaplugin.net/category/demo-free-version/" target="_blank">Demo plugin free version</a>
Demo InstagrmWP plugin by instagrmwp.com <a href="http://instagrmwp.com/category/demo-free-version/" target="_blank">Demo plugin free version</a> 

= Features : =

* Easy settings 3 step Niceforms style.
* Short code genarete.
* Supports short code integration.
* Tab color.
* Count Follow Following Media on tab
* Biography and link site.
* List photo polaroid style
* Tooltip caption
* Images size Small width 150 px height 150 px, Middle width 220 px  height 220 px, Large width 100% height auto.
* Count likes Comments Userinphoto.
* Link to instagram page.
* Zoom lightbox photo.
* Support Browser IE8+ Safari5+ Opera12.16+ Firefox24+ Chrome29+.
* Support Mobile Tablet Ipad Notebook Desktop (default browser).

== Installation ==

1. Unzip and Upload all files to a sub directory in "/wp-content/plugins/".
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Click left menu name InstagrmWP to setting
